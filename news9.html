10.20.2025 UPDATES

Version 4.0.0 Patch Notes

Maintenance to update the game will start at approx. 11:00 p.m. on September 24, 2025 (PT). This game will be updated to Ver.4.0.0 Once the update is released, the app will need to be updated in order to play. Please check that you have a stable network connection before updating the app. (Wi-Fi recommended)

Reduction of App Memory Use

We will be introducing a feature and other improvements to reduce the memory used by the game in the Ver.4.1.0 update ahead of the MYSEKAI release that was announced.

  • Managing the amount of Song Version data saved
  • Song Version Files will be added to File Management in Options so that the amount of Song Versions saved to device can be managed.
  • Players can choose from the following options: Never Save, 50, 100, 200, and Always Save. When the number of saved files exceed the specified number, old Song Version Files will be deleted sequentially.
  • 100 songs will be the default number of versions saved.
  • On the first login after the update, data that exceed the 100 song limit will be deleted. As a result, your login time may take longer than usual.

Other Updates

  • Addition of external payment banner.
  • Size of the message that appears when a user is banned is set to 350 bytes.
  • Update to support API level 35 for Google Play apps.
  • Update Google Play Billing Library to Ver.7.0
  • Android version only: The required space for the master data download will now be shown (This was already applied to iOS as of Ver.4.0.1)
